## LL(1)分析法

### 文法定义：

判断表达式的文法

$$
G[E]:\\
E->IR|AIR\\
R->\epsilon|AIR\\
I->FO\\
O->\epsilon|MFO\\
F->b|n|(E)\\
A->+|-\\
M->*|/\\
$$

判断条件表达式的文法
$$
G[E]:\\
S-> if\ T\ then\\
T->EPE|odd E\\
E->IR|AIR\\
R->\epsilon|AIR\\
I->FO\\
O->\epsilon|MFO\\
F->b|n|(E)\\
A->+|-\\
M->*|/\\
P->=|\#|<|<=|>|>=
$$
